
### 方法一 wait & notify
```java
    static int state = 1;

    public static void main2(String[] args) {

        Object obj1 = new Object();

        new Thread(new Runnable() {
            @Override
            public void run() {
                while (true) {

                    try {
                        synchronized (obj1) {
                            if (state != 1) {
                                obj1.wait();
                            }
                            System.out.println("1");
                            state = 2;
                            obj1.notify();
                        }
                    } catch (Exception e) {
                        e.printStackTrace();
                    }
                }

            }
        }).start();

        new Thread(new Runnable() {
            @Override
            public void run() {

                while (true) {
                    try {

                        synchronized (obj1) {
                            if (state != 2) {
                                obj1.wait();
                            }
                            System.out.println("2");
                            state = 1;
                            obj1.notify();
                        }
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                }

            }
        }).start();
    }
```

### 方法二 倒数锁
```java
static CountDownLatch c1 = new CountDownLatch(0);
    static CountDownLatch c2 = new CountDownLatch(1);

    /**
     * 使用倒数锁
     */
    public static void main3(String[] args) {


        new Thread(new Runnable() {
            @Override
            public void run() {
                try {
                    while (true) {
                        c1.await();
                        System.out.println(1);
                        c1 = new CountDownLatch(1);
                        c2.countDown();
                    }

                } catch (Exception e) {
                    e.printStackTrace();
                }
            }
        }).start();

        new Thread(new Runnable() {
            @Override
            public void run() {
                try {
                    while (true) {
                        c2.await();
                        System.out.println(2);
                        c2 = new CountDownLatch(1);
                        c1.countDown();
                    }

                } catch (Exception e) {
                    e.printStackTrace();
                }
            }
        }).start();
    }
```

### 方法三  信号量
可以方便的扩展成1122等形式
```java
private static Semaphore s1 = new Semaphore(1);
    private static Semaphore s2 = new Semaphore(1);

    public static void main5(String[] args) throws Exception {

        s2.acquire();
        new Thread(new Runnable() {
            @Override
            public void run() {
                try {
                    while (true) {
                        s1.acquire();
                        System.out.println(1);
                        s2.release();
                    }
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
        }).start();

        new Thread(new Runnable() {
            @Override
            public void run() {
                try {

                    while (true) {
                        s2.acquire();
                        System.out.println(2);
                        s1.release();
                    }
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
        }).start();
    }
```

###第四 反例 可重入锁

程序运行报 `IllegalMonitorStateException` 是因为 unlock() 必须在持有锁的线程中操作 否则就会抛异常

```java
static ReentrantLock lock1 = new ReentrantLock();
    static ReentrantLock lock2 = new ReentrantLock();
    public static void main(String[] args) {

        lock2.lock();
        new Thread(new Runnable() {
            @Override
            public void run() {
                while (true) {
                    if (lock1.isLocked()){
                        continue;
                    }
                    System.out.println("in 1");
                    lock1.lock();
                    System.out.println(1);
                    if (lock2.isLocked()){
                        lock2.unlock();
                    }
                }
            }
        }).start();
        new Thread(new Runnable() {
            @Override
            public void run() {
                while (true) {
                    if (lock2.isLocked()){
                        continue;
                    }
                    lock2.lock();
                    System.out.println(2);
                    lock1.unlock();
                }
            }
        }).start();

    }
```
