### 概述

本项目是一个用vue实现的Admin后台，地址：[传送门](https://gitee.com/yluoyu/Admin-vue)
基于一个开源项目 [AG-Admin-UI](https://gitee.com/geek_qi/AG-Admin-v2.0),自己修改用来做学习Vue,同时作为demo，方便以后拿来直接使用。
同时 还有一个后台项目提供API接口-[bootdemo](https://gitee.com/yluoyu/bootdemo)

### 目录结构
```shell
├── build                      // 构建相关  
├── config                     // 配置相关
├── src                        // 源代码
│   ├── api                    // 所有请求
│   ├── assets                 // 主题 字体等静态资源
│   ├── components             // 全局公用组件
│   ├── directive              // 全局指令
│   ├── filtres                // 全局filter
│   ├── mock                   // mock数据
│   ├── router                 // 路由
│   ├── store                  // 全局store管理
│   ├── styles                 // 全局样式
│   ├── utils                  // 全局公用方法
│   ├── view                   // view
│   ├── App.vue                // 入口页面
│   └── main.js                // 入口 加载组件 初始化等
├── static                     // 第三方不打包资源
│   ├── jquery
│   └── Tinymce                // 富文本
├── .babelrc                   // babel-loader 配置
├── eslintrc.js                // eslint 配置项
├── .gitignore                 // git 忽略项
├── favicon.ico                // favicon图标
├── index.html                 // html模板
└── package.json               // package.json

```
